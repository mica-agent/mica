# Mica

An extensible canvas runtime for building products with AI. Mica connects to your existing git repos and provides a surface where humans and AI agents collaborate — from first idea through deployed reality.

## Quick Start

```bash
npm install
npm run dev
```

The dev server starts the Vite frontend and the Express backend (port 3001).
