# Card Class: text

Displays plain text content in a preformatted block.

## Rendering
Renders content inside a `<pre class="card-text">` element with HTML entity escaping. No formatting or editing capabilities.

## Interactions
Read-only display. Automatically refreshes when its file changes via `file-changed` event listener.

## Data Format
Primary file: `content.txt` -- Plain text.

## Dependencies
None.
