Versioning

Each layer has its own version metaphor, native to its geological character. Versions stack like strata along the edge of the canvas.

Layer-Native Metaphors

Fossilize

Mission versions are preserved as geological records in version strata. Each fossilized mission is an immutable layer of sediment, readable from above.

Crystallize

Experience versions crystallize: edges sharpen, surfaces refine, the design slightly elevates as it solidifies from fluid exploration to fixed form.

Bedrock

Architecture versions shift down to become bedrock: foundational, load-bearing, the stable ground upon which implementation is built.

Commit

Implementation uses standard git-style commits. Familiar, concrete, immediate. The most granular versioning in the stack.
